# 陶瓷AI视觉工厂 — 完整产品规划

---

## 1. 产品愿景

**一句话：** 让景德镇每一个陶瓷卖家，用手机拍一张照片，3分钟拿到专业级电商视觉素材，成本从500元降到5元。

**补充定位：** 这不是一个通用AI图片工具，而是景德镇陶瓷电商垂直场景的"视觉内容工厂"——深度理解陶瓷材质（青花、粉彩、建盏、白瓷的光影差异）、电商规范（淘宝800×800主图、拼多多白底图、抖音竖版视频封面）、景德镇审美（传统与现代融合的视觉语言）。

## 2. 核心功能清单

### P0 — MVP必做（没有这些产品不成立）

| 功能 | 输入 | 输出 | 用户价值 | 技术实现 |
|------|------|------|----------|----------|
| **AI产品主图生成** | 1-3张实物照片（手机拍摄即可） | 6张不同场景的800×800电商主图（白底/木桌/茶席/展厅/生活场景/节日主题） | 替代500-2000元的产品图拍摄，3分钟出图 | ComfyUI + ControlNet + IP-Adapter，陶瓷材质LoRA |
| **一键换背景** | 产品抠图 + 选择场景模板 | 产品融入新场景的合成图 | 同一产品快速适配不同店铺风格 | SAM2抠图 + 场景融合Inpainting |
| **智能抠图** | 实物照片 | 透明背景PNG（保留陶瓷反光和阴影细节） | 电商白底图需求，比Remove.bg更懂陶瓷 | SAM2 + 陶瓷边缘精修后处理 |
| **尺寸规格适配** | 选择目标平台 | 自动裁剪/扩展为平台规范尺寸 | 一张原图适配淘宝/拼多多/抖音/1688 | 智能裁剪 + Generative Expand |

### P1 — 核心差异化（第2-3个月）

| 功能 | 输入 | 输出 | 用户价值 | 技术实现 |
|------|------|------|----------|----------|
| **AI详情页生成** | 产品名称 + 卖点关键词 + 产品图 | 完整详情页长图（平台规范宽度750px） | 替代200-800元的详情页制作，从2天变10分钟 | LLM文案生成 + 模板引擎 + 图文混排 |
| **多风格渲染** | 产品图 + 选择风格（现代简约/中式古典/日式侘寂/欧式奢华） | 不同视觉风格的产品展示图 | 同一产品覆盖不同审美客群 | 风格LoRA + ControlNet保持产品形态 |
| **文字排版引擎** | 产品文案 + 详情页模板 | 带专业排版的详情页（含标题/卖点/参数/售后） | 非设计人员也能产出专业排版 | Canvas/SVG渲染引擎 + 字体库 |
| **素材收藏夹** | 浏览/搜索素材 | 个人素材收藏集 | 建立用户粘性，素材复用 | MongoDB用户态存储 |

### P2 — 增长引擎（第4-6个月）

| 功能 | 输入 | 输出 | 用户价值 | 技术实现 |
|------|------|------|----------|----------|
| **景德镇专属素材库** | — | 5000+ 陶瓷场景/纹理/器型素材 | 本地化素材是竞品无法复制的护城河 | 线下拍摄 + AI生成扩充 |
| **AI素材定制** | 文字描述（如"青花瓷碗在竹编茶席上，晨光"） | 定制场景图 | 无限扩展素材可能性 | SDXL/Flux + 陶瓷场景Prompt工程 |
| **批量处理** | 产品图片文件夹（10-50张） | 批量生成主图/抠图/换背景 | 大卖家效率需求 | 任务队列 + 并行推理 |
| **团队协作** | 邀请团队成员 | 多人共享素材库和项目 | 有团队的中型卖家 | RBAC权限 + 共享空间 |
| **API接口** | REST API调用 | 程序化调用所有能力 | SaaS大客户、ERP系统集成 | NestJS API Gateway |

## 3. 用户旅程

### 首次体验旅程（注册到第一张图，目标 < 5分钟）

```
[1] 触达
    ├─ 抖音/小红书看到对比视频（手机拍的 vs AI生成的）
    ├─ 景德镇本地地推/陶瓷市场摊位扫码
    └─ 淘宝卖家群口碑推荐
         │
[2] 注册（30秒）
    ├─ 手机号 + 验证码（复用Venux Cloud认证）
    ├─ 新用户赠送 20次免费额度
    └─ 选择身份：卖家/设计师/经销商
         │
[3] 引导（60秒）
    ├─ 3步新手引导动画
    │   ① 拍照tips（光线/角度/背景）
    │   ② 选择场景风格
    │   ③ 一键生成
    └─ 提供3张示例图可直接体验（无需上传）
         │
[4] 核心操作（120秒）
    ├─ 上传照片（支持相册/拍照）
    ├─ AI自动识别产品类型（碗/瓶/盘/壶/杯...）
    ├─ 推荐最佳场景模板（基于产品类型）
    ├─ 点击"生成"→ 等待30-60秒
    └─ 展示6张结果图
         │
[5] 结果处理（60秒）
    ├─ 预览大图 / 对比原图
    ├─ 选择满意的图片
    ├─ 一键下载（免费额度内带轻水印，付费去水印）
    └─ 分享到微信（裂变入口）
         │
[6] 留存钩子
    ├─ "再生成一套详情页？" → 引导P1功能
    ├─ "收藏这个场景，下次直接用"
    └─ 推送：每日免费素材更新
```

### 回头客旅程（日常使用）

```
打开App → 最近项目 → 选择产品 → 选新场景/改文案 → 生成 → 下载 → 上架
                                  ↑
                           素材库浏览 → 发现新场景 → 应用到已有产品
```

## 5. 竞品分析

### 直接竞品

| 产品 | 优势 | 劣势 | 我们的差异化 |
|------|------|------|-------------|
| **千鹿AI**（国内） | 电商AI图片功能齐全，用户量大 | 通用型，不懂陶瓷；无详情页功能；素材库泛化 | 陶瓷垂直深度 > 通用广度 |
| **美间AI**（国内） | 家居场景强，渲染质量高 | 家居为主，不覆盖陶瓷；定价高（¥299/月起） | 陶瓷专属场景+亲民定价 |
| **Photoroom**（海外） | 抠图质量行业标杆，App体验好 | 无中国电商尺寸适配；无详情页；月费$12.99偏高 | 中国电商全平台适配+本地化 |
| **Flair.ai**（海外） | AI产品图生成质量高 | 英文为主，无中国场景；无素材库 | 景德镇专属素材+中文优先 |
| **ZMO.ai**（海外） | 模特换装+产品图，功能丰富 | 通用型；中国电商适配弱 | 陶瓷场景深度定制 |

### 间接竞品

| 产品 | 威胁程度 | 应对策略 |
|------|----------|----------|
| **Canva** | 中 | 他们做通用设计，我们做陶瓷垂直；合作可能 > 竞争 |
| **稿定设计** | 中 | 类似Canva，但电商模板多；我们用AI生成替代模板选择 |
| **美图秀秀** | 低 | 面向C端，电商专业度不够 |
| **淘宝官方工具** | 高（长期） | 如果淘宝推出内置AI工具，我们需要做到体验远超；但淘宝工具不会做垂直场景 |

### 核心竞争壁垒

```
1. 场景壁垒：景德镇线下拍摄5000+陶瓷专属场景素材（竞品无法短期复制）
2. 模型壁垒：陶瓷材质LoRA（青花/粉彩/建盏/白瓷分别训练），生成质量远超通用模型
3. 数据壁垒：积累的陶瓷产品图生成数据，持续优化模型
4. 渠道壁垒：景德镇本地地推关系网，线下信任 > 线上广告
5. 生态壁垒：详情页+主图+素材一站式，替代多个工具
```

## 6. MVP范围

### MVP定义：能完成"上传照片 → 生成6张电商主图"的核心闭环

**包含功能：**
- ✅ 手机号注册/登录
- ✅ AI产品主图生成（3个基础场景：白底/木桌/茶席）
- ✅ 智能抠图（SAM2）
- ✅ 尺寸适配（淘宝800×800、拼多多800×800、抖音750×1000）
- ✅ 图片下载（带水印/去水印）
- ✅ 免费额度系统（20次/月）
- ✅ 付费订阅（基础版¥49/月）

**不包含（V1.1+）：**
- ❌ AI详情页生成
- ❌ 多风格渲染
- ❌ 素材库浏览
- ❌ 批量处理
- ❌ 团队协作

### 开发周期预估

| 阶段 | 时间 | 产出 |
|------|------|------|
| **Week 1-2** | 基础架构 | NestJS项目搭建、MongoDB schema、ComfyUI环境配置、前端Vue3脚手架 |
| **Week 3-4** | AI核心 | ComfyUI workflow搭建、陶瓷产品图LoRA训练/微调、ControlNet配置、SAM2集成 |
| **Week 5-6** | 前端开发 | 上传/预览/下载页面、场景选择UI、生成进度展示 |

| **Week 11-12** | 上线运营 | 部署到Venux Cloud、地推准备、种子用户邀请测试 |

**总计：12周（3个月），1人全栈开发**

**资源消耗预估：**
- GPU服务器（ComfyUI推理）：A10 24GB × 1，约¥3000/月（Venux Cloud自有成本价更低）
- 存储：500GB起，约¥200/月
- 域名+CDN+短信：约¥500/月
- 总月运营成本：<¥5,000

### AI模型选型

| 用途 | 模型方案 | 理由 | 显存需求 |
|------|----------|------|----------|
| **产品图生成** | SDXL + IP-Adapter + ControlNet (Depth/Canny) | SDXL质量好，ControlNet保持产品形态，IP-Adapter保持产品一致性 | 12-16GB |
| **陶瓷材质增强** | 自训练LoRA（青花/粉彩/建盏/白瓷各一个） | 通用模型对陶瓷光影表现差，LoRA微调提升质感 | 同上 |
| **抠图** | SAM2 (Segment Anything 2) | 开源SOTA，支持自动和点击交互 | 8GB |
| **背景生成/融合** | SDXL Inpainting | 产品融入场景的自然度 | 12GB |
| **文案生成** | Qwen2.5-72B (API) 或 DeepSeek-V3 (API) | 中文文案质量好，API调用省GPU | 无需本地GPU |
| **详情页排版** | Canvas渲染引擎 + 预设模板 | 确定性输出，不需要AI | CPU即可 |

### 存储方案

| 数据类型 | 存储方案 | 预估用量（首年） | 成本 |
|----------|----------|-----------------|------|

|------|--------|------|

## 8. 增长策略

### 前1000个用户的获取路径

#### Phase 1：种子用户（0-100人，Month 1-2）

**渠道：景德镇本地地推**

| 动作 | 具体执行 | 预期转化 |
|------|----------|----------|
| 陶瓷市场驻点 | 每周3天在景德镇陶瓷市场/陶溪川摆摊，现场演示"拍照→出图" | 每天接触20人，转化5人注册 |
